【步驟 1】下載資料
--------------------------------------------------------------------------------
  從以下連結下載專案壓縮檔（實際連結以貴單位發佈為準）：
  https://drive.google.com/file/d/1JLcY68mQ9ywcCV7PTWDqF7_7c290vHkv/view?usp=drive_link


【步驟 2】解壓縮
--------------------------------------------------------------------------------
  將下載的檔案解壓縮，可使用 7-Zip 或 Windows／Mac 內建解壓縮工具。
  建議解壓到「桌面」或您方便找到的位置（例如：CCTV_MVP 資料夾）。


【步驟 3】安裝 Node.js
--------------------------------------------------------------------------------
  官方下載（建議 LTS）：
  https://nodejs.org/zh-tw/download

  下載後依安裝精靈指示完成安裝。

  （選用）若需與內部文件相同版本，可改用下列方式安裝 Node.js 24.x：

  ■ Mac／Linux（nvm）
    在終端機依序執行（忽略#後內容並一行一行輸入）：

    # 安裝 nvm
    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash

    # 重新開啟終端機後，先載入：
    source "$HOME/.nvm/nvm.sh"

    # 安裝 Node.js
    nvm install 24


  ■ Windows（Chocolatey，需管理員權限 PowerShell）

    # 下載並安裝 Chocolatey：
    powershell -c "irm https://community.chocolatey.org/install.ps1|iex"

    # 下載並安裝 Node.js：
    choco install nodejs-lts --version="24"

    # 核對 Node.js 版本：
    node -v # 應會印出 "v24.14.1"。

    # Verify the Node.js version:
    node -v # Should print "v24.14.1".

    # 核對 npm 版本：
    npm -v # 應會印出 "11.11.0"。



【步驟 4】開啟終端機
--------------------------------------------------------------------------------
  Mac
    方法一：桌面按 Command + F，搜尋「終端機」
    方法二：啟動台 → 其他 → 終端機

  Windows
    方法一：Win + X → 選擇「終端機」或「Windows Terminal」
    方法二：開始選單搜尋「cmd」或「PowerShell」


【步驟 5】複製專案資料夾路徑
--------------------------------------------------------------------------------
  找到【步驟 2】解壓縮後的 CCTV_MVP（或專案）資料夾，選取後：

    Mac：     Option + Command + C（複製路徑）
    Windows： Shift + Ctrl + C（於檔案總管網址列或「複製路徑」依系統版本操作）

  若無快捷鍵，可將資料夾拖入終端機視窗，通常會自動帶入路徑。


【步驟 6】在終端機進入專案目錄
--------------------------------------------------------------------------------
  輸入 cd 並貼上路徑（中間空一格），例如：

    cd /Users/您的帳號/Desktop/CCTV_MVP

  Windows 範例：

    cd C:\Users\您的帳號\Desktop\CCTV_MVP


【步驟 7】確認環境
--------------------------------------------------------------------------------
  終端機輸入：

    node -v
    npm -v

  兩者皆應顯示版本號。若沒有，請回到【步驟 3】檢查 Node.js 是否安裝成功。


【步驟 8】下載專案依賴套件
--------------------------------------------------------------------------------
    npm install


【步驟 9】啟動開發伺服器
--------------------------------------------------------------------------------
    npm run dev


【步驟 10】用瀏覽器開啟
--------------------------------------------------------------------------------
  終端機會顯示類似 http://localhost:5173/ 的網址（埠號以實際輸出為準）。
  請複製該網址，在 Chrome、Edge、Safari 等瀏覽器網址列貼上並開啟。


--------------------------------------------------------------------------------
  同區網手機／平板（選用）
--------------------------------------------------------------------------------
  若要在同一 Wi‑Fi 下用其他裝置開啟：
  1. 查電腦的區網 IP（例如 192.168.x.x）。
  2. 將網址中的 localhost 改成該 IP，例如 http://192.168.1.100:5173/
  3. 若無法連線，請檢查防火牆是否允許該連接埠。

  若僅需離線瀏覽列表／CSV 示範，也可直接雙擊專案內的 demo.html（無需 npm）。
